# Data Science Summit 2026 - Frontend Website

This project is a frontend-only website for the **Data Science Summit 2026** event.

## Project Overview

The website is designed to provide attendees and participants with all important event information in a clean and user-friendly format.

## Pages and Features

### 1. Landing Page
- Intro section for Data Science Summit 2026
- Event highlights and key call-to-actions

### 2. Speakers
- Featured speaker:
	- **Siddharth Samant** - **Red Hat**

### 3. Sponsors
- Dedicated section/page for event sponsors
- Space to showcase sponsor logos and details

### 4. Events
- Event listing with:
	- Description
	- Date
	- Time
	- Venue
	- Registration link (Unstop)

### 5. Contact Us
- Contact section/form for inquiries and communication

### 6. Teams
- Team-wise structure
- Members per team
- Faculty advisor details

## Tech Scope

- Frontend only (no backend integration)

## Goal

To build an informative, modern, and accessible event website for Data Science Summit 2026.
